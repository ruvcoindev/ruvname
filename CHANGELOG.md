# Changelog

Все изменения в `ruvname` документируются здесь в соответствии с [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

## [v0.1.0] - 2025-08-15
### Added
- Добавлена поддержка DNS-over-HTTPS (DoH)
- Реализован встроенный веб-интерфейс (GUI)
- Поддержка P2P-сети на IPv6 и IPv4
- Поддержка доменов: `.ruv`, `.mesh`, `.node`, `.p2p`, `.tamb`, `.tmb`, `.dweb`, `.dht`, `.hub`
- Реализован майнинг доменов (PoW)
- Добавлены мультиархитектурные сборки: Linux, Windows, macOS
- Добавлен `Makefile` для упрощённого запуска
- Добавлена документация API и `mdbook`

### Changed
- Переработана структура `ruvname.toml`
- Улучшена стабильность DNS-резолвера
- Обновлены зависимости: `cross`, `web-view`, `rustls`

### Fixed
- Исправлены утечки памяти в P2P-сети
- Исправлены ошибки валидации блоков
- Исправлена работа с ключами на Windows
- Устранена ошибка `GLIBC_2.28 not found` при сборке

### Removed
- Удалена поддержка устаревших фич
- Удалён ненужный `build.rs` (упрощена сборка)
