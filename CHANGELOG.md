# Changelog

Все изменения в `ruvname` документируются здесь в соответствии с [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

## [v0.1.0] - 2025-08-15
### Added
- Добавлена поддержка DNS-over-HTTPS (DoH)
- Добавлен встроенный веб-интерфейс (GUI)
- Реализована P2P-сеть на основе блокчейна
- Поддержка мультиплатформенной сборки (Linux, Windows, macOS)

### Changed
- Переработана структура конфигурации `ruvname.toml`
- Улучшена стабильность DNS-резолвера

### Fixed
- Исправлены утечки памяти в P2P-сети
- Исправлены ошибки валидации блоков

### Removed
- Удалена поддержка устаревших фич

## Как использовать
- `Unreleased` — изменения, ещё не включённые в релиз
- Теги вида `[v0.1.0]` — версии, соответствующие тегам в Git
