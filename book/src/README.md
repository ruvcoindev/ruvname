# 🌐 RUVNAME

**RUVNAME** — децентрализованная система доменных имён (DNS), построенная на блокчейне.  
Поддерживает домены: `.ruv`, `.mesh`, `.node`, `.p2p`, `.dweb`, `.tmb`, `.tamb` и другие.

> 🔐 Безопасно | 🌐 Децентрализовано | 💻 Поддержка Linux, Windows, macOS
