RUVNAME Documentation

🌐 RUVNAME

RUVNAME — децентрализованная система доменных имён (DNS), построенная на блокчейне.
Поддерживает домены: .ruv, .mesh, .node, .p2p, .dweb, .tmb, .tamb и другие.

🔐 Безопасно | 🌐 Децентрализовано | 💻 Поддержка Linux, Windows, macOS

🚀 Быстрый старт

1. Скачайте

👉 Releases

  • Linux: ruvname-linux-amd64-v0.1.0-nogui.deb
  • Windows: ruvname-windows-x64-v0.1.0-gui.zip
  • macOS: ruvname-macos-universal-v0.1.0-nogui.dmg

2. Установите

Linux

sudo dpkg -i ruvname-linux-amd64-v0.1.0-nogui.deb
sudo systemctl enable ruvname
sudo systemctl start ruvname

🔧 Конфигурация

Файл: /etc/ruvname.conf (Linux) или ruvname.toml (Windows)

Пример

[net]
peers = ["[fa00:4715:66fa:2bd:3032:b5d1:e86f:239c]:6890"]
listen = "[::]:6890"
public = true

[dns]
listen = "127.0.0.1:5311"
forwarders = ["https://dns.adguard.com/dns-query"]
bootstraps = ["9.9.9.9:53", "94.140.14.14:53"]

[mining]
threads = 0
lower = true

🌐 DNS API

GET /resolve

Разрешает доменное имя.

Пример

GET /resolve?name=ya.ru.bit&type=A

🔐 Домены API

POST /domain

Регистрирует новый домен (запускает майнинг).

Тело запроса

{
  "name": "mydomain",
  "zone": "ruv",
  "data": {
    "records": [
      {
        "type": "A",
        "domain": "mydomain.ruv",
        "addr": "127.0.0.1",
        "ttl": 300
      }
    ],
    "info": "My personal domain",
    "contacts": [
      {
        "name": "email",
        "value": "me@example.com"
      }
    ]
  }
}

📦 Блокчейн API

GET /blocks

Возвращает последние блоки.

Параметры

  • limit (int, опционально) — количество, по умолчанию 10
  • offset (int, опционально) — смещение

Ответ

[
  {
    "height": 150,
    "hash": "a1b2c3...",
    "timestamp": 1723700000,
    "tx_count": 1,
    "difficulty": 24
  }
]

🟨 JavaScript Пример

<!DOCTYPE html>
<html>
<head>
  <title>RUVNAME JS Client</title>
</head>
<body>
  <button onclick="getStatus()">Получить статус</button>
  <pre id="output">...</pre>
  <script>
    function getStatus() {
      fetch("http://127.0.0.1:5311/status")
        .then(r => r.json())
        .then(data => {
          document.getElementById("output").textContent = JSON.stringify(data, null, 2);
        });
    }
  </script>
</body>
</html>
